### create new modules
开发说明：
1、均在modules文件夹下进行活动开发
2、通用控件写入components目录下
3、通用脚本写在javascripts目录下
4、每个module下必须有一个index.js文件作为入口文件
5、module文件夹名称即为打包后的html文件名称及相关资源路径的名称
